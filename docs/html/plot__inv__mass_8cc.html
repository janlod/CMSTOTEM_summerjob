Plot histograms after applying various cuts (pt, eta, dxy, chi2, rho).

Loops over the cuts vector and draws each histogram on the same canvas with different colors.

Parameters
filepathPath to the ROOT file with initial data.
cut_optionCut type: "pt", "eta", "dxy", "chi2", or "rho".
initial_kin_cutsInitial kinematic cuts to apply.
initial_chi2cutsInitial chi2 cuts to apply.
initial_rho_cutInitial rho cut to apply.
cutsVector of cut thresholds to scan.
nbinsNumber of histogram bins.
minMinimum histogram value.
maxMaximum histogram value.
outpathPath to save the resulting combined histogram canvas.

Plot the invariant mass of 4-track combinations assuming a common mass.

Reconstructs the invariant mass using the provided mass hypothesis (massparam), drawing a vertical reference line at 2220 MeV.

Parameters
treePointer to a TTree containing branches: trk_pt, trk_eta, trk_phi, ntrk.
